# infotecs-react
## Тестовое задание для стажера React Frontend-разработчика
## Инструкция по запуску проекта
### Установка и запуск

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/artem5-5/infotecs-react
   ```

2. **Установка зависимостей**
  ```bash
   npm install

   или

   yarn install
   ```

3. **Запуск в режиме разработки**
  ```bash
    npm run start

    или

    yarn start
  ```

4. **Сборка**
  ```bash
    npm run build

    или

    yarn build
  ```

### Скрипты проекта

- `start` - запуск development сервера
- `build` - сборка проекта для production
- `preview` - запускает сборку React-приложения
- `deploy` - используется для размещения проекта на сервисе GitHub Pages

### Основной стек

- TypeScript
- Webpack + Npm
- React 16
- Ant design 5 и styled-components
- React router 6
- TanStack Query 4 + axios
- dayjs
- Git (github.com)
- Архитектура FSD (Feature-Sliced Design)

### Выполненные задачи:

Реализован проект базы пользователей с использованием следующего стека технологий. Проект содержит основные компоненты:
- Страница авторизации
- Страница 404
- Страница пользователей
- Модальное окно редактирования пользователя
- Модальное окно создания пользователя
